## About me

Backend engineer based in Malta, focused on building reliable, observable and high-performance backend systems.

I have strong experience with Python backend development, asynchronous processing, database design, service architecture and production systems.  
Currently learning and actively exploring Rust for performance-critical infrastructure, networking and developer tooling.

- Backend engineering with Python for 15+ years
- Experience with Django, FastAPI, LiteStar and async Python
- Working with PostgreSQL, ClickHouse, Redis, MongoDB and SQLAlchemy
- Interested in Rust, distributed systems, observability and high-load APIs
- Currently working at [Altenar](https://altenar.com)
